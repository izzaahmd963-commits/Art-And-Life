# Art-And-Life
A responsive website celebrating art and life with interactive features.

## Features
- Fixed navigation bar with blur effect
- Mobile-responsive hamburger menu
- Auto-playing image slideshow
- Accordion menu
- Form validation
- Fully responsive design

## Technologies
- HTML5
- CSS3
- JavaScript
- Font Awesome 6
- Google Fonts
- Unsplash Images
- create one file all code already save index.html
